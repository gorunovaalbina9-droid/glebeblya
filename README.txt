ПРОСТАЯ ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА
=================================

Этот файл объясняет, что именно нужно сделать шаг за шагом.

1. УСТАНОВИТЬ PYTHON (ЕСЛИ ЕГО НЕТ)
----------------------------------

1. Открой браузер и зайди на сайт: https://www.python.org/downloads/
2. Скачай последнюю версию для Windows.
3. При установке ВАЖНО: поставь галочку "Add Python to PATH".
4. Дальше просто жми "Next" / "Install".


2. ПОЛУЧИТЬ ТОКЕН БОТА В TELEGRAM
---------------------------------

1. Открой Telegram.
2. Найди пользователя @BotFather.
3. Напиши ему /start.
4. Напиши команду /newbot.
5. Следуй инструкциям (название бота, юзернейм).
6. В конце он пришлёт строку вида:

   1234567890:AA...очень_длинная_строка...

   Это и есть ТОКЕН. Сохрани его.

7. Отключи privacy mode (чтобы бот видел сообщения в группе):
   - в @BotFather напиши /setprivacy
   - выбери своего бота
   - выбери Disable


3. ВСТАВИТЬ ТОКЕН В ФАЙЛ bot.py
--------------------------------

1. В папке c:\Users\Альбина\глебебля2 должен лежать файл bot.py (я его уже создал).
2. Открой этот файл в редакторе (Cursor, VS Code или Блокнот).
3. Найди строку:

   BOT_TOKEN = "PASTE_YOUR_BOT_TOKEN_HERE"

4. Заменить текст внутри кавычек на твой реальный токен, например:

   BOT_TOKEN = "1234567890:AAABBBССС..."

5. Сохрани файл.


4. ЗАПУСТИТЬ КОМАНДЫ В POWERSHELL
---------------------------------

1. Открой PowerShell.
2. Введи команду (и нажми Enter):

   cd "c:\Users\Альбина\глебебля2"

3. Создай виртуальное окружение (один раз):

   python -m venv venv

4. Активируй виртуальное окружение (делай это каждый раз, когда хочешь запускать бота):

   .\venv\Scripts\Activate

   Если всё хорошо, слева в строке появится (venv).

5. Установи нужные библиотеки (один раз):

   pip install -r requirements.txt

6. Запусти бота:

   python bot.py

7. В окне PowerShell должно появиться сообщение:

   Бот запускается. Нажмите Ctrl+C для остановки.


5. ПРОВЕРКА РАБОТЫ БОТА
-----------------------

1. Открой Telegram, найди своего бота по юзернейму.
2. Напиши ему в ЛИЧКУ команду:

   /start

   Он должен ответить приветствием.

3. Добавь бота в нужный групповой чат.
4. В этом групповом чате отправь несколько медиа (фото, видео, документ).
5. В этом же чате напиши команду:

   /resend_media

   Бот должен начать пересылать все сохранённые медиа в этот чат.

6. Если хочешь очистить его память по этому чату, напиши:

   /clear_media


6. КАК ОСТАНОВИТЬ БОТА
----------------------

В окне PowerShell, где запущен бот, нажми:

   Ctrl + C

Бот остановится.


Если на каком-то шаге появится ошибка в PowerShell или бот не отвечает — просто пришли сюда текст ошибки, и я помогу разобрать, что не так.
